# Мониторинг 3х фазной сети на ардуино

Этот проект нужен для наблюдения за показателями 3 фазной электросети. Он помогает распределить и контролировать нагрузку по фазам питания. 

У меня на [канале](https://www.youtube.com/channel/UCxnOUs7_LihbzWgDb2nf0ew) есть [видео](https://www.youtube.com/channel/UCxnOUs7_LihbzWgDb2nf0ew) как повторить проект.
